*************
ガイドツアー
*************

このセクションはSageについてのガイドツアーです。
いくつかの例を見るには"Sage Constructions"を見てください。
網羅的な例を探すには"sage Reference Manual"を参照して下さい。
Sageノートブックを用いる場合、 ``Help`` リンクをクリックする事で、インタラクティブに操作できます。

(このチュートリアルをSageノートブックから見ている場合、 ``shift-enter`` を押すことでインプットセルを評価する事が出来ます。
shift-enterを押す前に編集する事もできます。
いくつかのMacでは、 ``shift-enter`` の代わりに ``shift-return`` を押す必要があります。)
