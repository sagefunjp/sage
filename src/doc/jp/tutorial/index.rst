Sage チュートリアルへようこそ！
===============================

Sageは無料でオープンソースな数式処理ソフトウェアで、代数、幾何、数論、暗号論、数値計算等での研究開発や教育を支援します。
Sageの開発モデルや、Sageで用いられる技術は、オープン性、コミュニティ、協調、協力に重きを置いている点で、他とは異なっています。
私達は、車輪の再発明をする事無く、車を作ろうとしています。
Sageの目標は、持続可能で、自由で、オープンソースな、Maple、Mathematica、Magma、MATLABの代替を作る事です。

このチュートリアルは数時間でSageに慣れ親しむ最良の方法です。
HTML、PDF、Sage ノートブック形式でチュートリアルで読む事ができます。
(Sageを起動し、 ``Help`` をクリックし、 ``Tutorial`` をクリックしてください。)
